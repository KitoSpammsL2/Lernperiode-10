Heute habe ich mit der Entwicklung meines Projekts Space Clicker in Kotlin begonnen. Zuerst habe ich ein neues Android-Projekt erstellt und die Entwicklungsumgebung eingerichtet. Anschließend habe ich die Grundfunktionen des Spiels programmiert, darunter eine Punkteanzeige, einen Klick-Button sowie ein Upgrade-System, das die Punkte pro Klick erhöht. Während der Entwicklung traten mehrere Probleme auf, insbesondere Abstürze der App beim Start. Nachdem diese Fehler behoben wurden, konnte die App erfolgreich gestartet werden. Das aktuelle Spiel ist funktionsfähig und bildet die Grundlage für weitere Features wie passives Einkommen und ein verbessertes Design.
